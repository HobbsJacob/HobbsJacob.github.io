Soil Sensors

For my 2017-2018 summer placement I worked at Intelligent Tek, a small company creating tech products for the Agriculture industry. I had a part in creating several prototype soil moisture sensors. The first used WiFi to transmit data from ESP32s placed in fields, and another variant used RF and ran on Arduino. I also created the desktop applications used to record data from these devices and upload it to the products web database.

Mobile Applications

One of the sensors we created used Bluetooth Low Energy to transmit data to the users phone. I created an iOS application which received, displayed data and allowed the user to upload it to the products database. The application was approved for distribution on the Apple Store. I also converted the existing Android application to use Bluetooth Low Energy and made various changes to the security of the Android application and the user authentication process, both on the app and the web server.
